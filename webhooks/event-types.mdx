## device.connected

A WhatsApp device successfully connected and authenticated.

<AccordionGroup>
  <Accordion title="Sample Request Body">
    ```json
    {
      "event": "device.connected",
      "instanceId": "1234567",
      "data": {
        "displayName": "John Doe",
        "phoneNumber": "+1234567890"
      },
      "timestamp": "2024-01-15T10:30:00.000Z"
    }
    ```
  </Accordion>
</AccordionGroup>

## device.disconnected

A WhatsApp device disconnected from the server.

<AccordionGroup>
  <Accordion title="Sample Request Body">
    ```json
    {
      "event": "device.disconnected",
      "instanceId": "1234567",
      "data": {
        "reason": "logged_out"
      },
      "timestamp": "2024-01-15T10:35:00.000Z"
    }
    ```
  </Accordion>
</AccordionGroup>

<AccordionGroup>
  <Accordion title="Sample Request Body">
    ```json
    {
      "event": "qr",
      "instanceId": "1234567",
      "data": {
        "qrCode": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA..."
      },
      "timestamp": "2024-01-15T10:25:00.000Z"
    }
    ```
  </Accordion>
</AccordionGroup>

<AccordionGroup>
  <Accordion title="Sample Request Body">
    ```json
    {
      "event": "qr.timeout",
      "instanceId": "1234567",
      "data": {},
      "timestamp": "2024-01-15T10:27:00.000Z"
    }
    ```
  </Accordion>
</AccordionGroup>

## message.received

<AccordionGroup>
  <Accordion title="Sample Request Body">
    ```json
    {
      "event": "message.received",
      "instanceId": "1234567",
      "data": {
        "messages": [
          {
            "id": "3EB0B430B6F8F1D0E053AC120E0A9E5C",
            "from": "1234567890@s.whatsapp.net",
            "content": "Hello, how are you?",
            "type": "text",
            "timestamp": 1705315800
          }
        ]
      },
      "timestamp": "2024-01-15T10:30:00.000Z"
    }
    ```
  </Accordion>
</AccordionGroup>
